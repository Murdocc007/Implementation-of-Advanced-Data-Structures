The Level_1 folder contains the following files :

LP5_Level1.java
Graph.java
Edge.java
Vertex.java
Report_Level1.pdf

In order to run the file we need to increase the stack space to 8mb

To execute the level 1 of LP5 run the following commands :
>> cd Level_1
>> javac *.java
>> java -Xss8m LP5_Level1 <path of the input file> [verbose]
